# 요구사항

실습을 위해서는 아래와 같은 것들을 설치해야 합니다. 각자 설치하고 오시기 바랍니다.

* [VirtualBox](http://virtualbox.org/) : 가상 머신을 만들 수 있는 도구
* [Vagrant](http://www.vagrantup.com) : VirtualBox에 가상 머신을 만들고 필요한 프로그램 설치 및 환경구성을 하는 도구

# 시작하기

## VM 생성, 환경 구성
```
  # vagrant up
```

## 생성된 VM에 접속하기
```  
  # vagrant ssh
```

## 생성한 VM 삭제
```
  # vagrant destroy
```
